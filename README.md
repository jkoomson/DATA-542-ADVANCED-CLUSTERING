## 📘 About This Repository

Welcome to the official student project repository for the **Advanced Clustering Methods** course in the Department of Mathematical Sciences, instructed by John Koomson.

This repository showcases student work focused on:

* Advanced clustering algorithms
* Unsupervised machine learning
* Statistical learning techniques
* Real-world data analytics
* Interactive data science applications

Projects explore datasets from domains such as:

💰 Finance & Insurance
🏥 Healthcare & Medicine
🧬 Biology & Genomics
🛒 Retail & Marketing
🌍 Environmental Science
🎓 Education Analytics

---

## 🚀 Project Objectives

Students were required to:

✅ Work with large real-world datasets
✅ Implement and compare multiple clustering methods
✅ Perform preprocessing and exploratory data analysis
✅ Evaluate clustering performance
✅ Interpret clusters in meaningful real-world contexts
✅ Produce professional reports and presentations
✅ Build reproducible analytical workflows in R

---

## 🧠 Clustering Methods Featured

The projects may include implementations of:

| Method                        | Description                     |
| ----------------------------- | ------------------------------- |
| k-Means                       | Partition-based clustering      |
| Hierarchical Clustering       | Tree-based clustering methods   |
| DBSCAN                        | Density-based clustering        |
| Gaussian Mixture Models (GMM) | Probabilistic clustering        |
| Spectral Clustering           | Graph-based clustering          |
| Self-Organizing Maps (SOM)    | Neural-network-based clustering |
| Kernel Methods                | Nonlinear clustering approaches |
